# Знакомство с CSS Grid

_Галина Григорьева, верстальщик HTML/CSS_

Галина рассказывает о CSS Grid: приводит описание основных терминов, CSS свойств с примерами вёрстки, а также ответами на часто задающиеся вопросы о том, возможно ли использовать эту технологию, если необходимо поддерживать IE11, а также может ли CSS Grid заменить Flexbox. 

Выступление будет полезно начинающим фронтенд-разработчикам.
